# Syntactic Monoid Explorer

A tool for computing the syntactic monoid of a DFA and demonstrating
the First Isomorphism Theorem for formal languages.

## Setup
pip install graphviz networkx matplotlib pandas pytest

## Run
python src/cli.py examples/parity_dfa.json
