Autómatas, expresiones regulares y monoides de transición

Visualizadores interactivos y análisis algebraico para los cursos de Teoría de la Computación y Matemática Discreta II. Toda la teoría sigue las convenciones del libro Introducción a la Teoría de la Computación de Rodrigo De Castro K. (Universidad Nacional de Colombia, 2024).

Construye y verifica un AFD, compila una expresión regular a AFN-λ (Teorema de Kleene Parte I) y al AFD mínimo (§2.16), simula una Máquina de Turing del modelo estándar (§6.1) sobre una cinta bidireccional, y conecta todo con el monoide de transición M(A) y el teorema de isomorfismo (Saracino §12).

Convenciones: AFD/AFN/AFN-λ, λ, ∪, □ Modelo MT: 6-upla (Q, q0, F, Σ, Γ, δ) Sin servidor: HTML + JS

¿Qué hace este proyecto?

Tres preguntas, una respuesta integrada:

  1. Construcción. Dada una expresión regular, compilarla a un AFN-λ y luego a un AFD mínimo, mostrando cada paso visualmente.
  2. Simulación. Editar y ejecutar una Máquina de Turing del modelo estándar de De Castro (§6.1) sobre una cadena de entrada, con la cinta bidireccional y la unidad de control animadas paso a paso.
  3. Análisis algebraico. Para un AFD dado, construir explícitamente su monoide de transición M(A), el homomorfismo natural φ : Σ* → M(A), y determinar si es un grupo, qué grupo, si es aperiódico, etc. Esto conecta directamente con Matemática Discreta II (clases de equivalencia, ℤ/nℤ, primer teorema del isomorfismo) y con el Teorema de Myhill-Nerode (§2.15) del libro.

Analizar tu propio AFD

El análisis algebraico (cálculo del monoide, idempotentes, tabla de Cayley, identificación del grupo) corre en Python y requiere instalación local. El flujo recomendado es: 

# 1) Clonar e instalar (Python 3.10+)
git clone <url-del-repo>
cd syntactic-monoid
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2) Definir tu AFD en JSON (ver examples/parity_afd.json)
# 3) Generar la hoja informativa en Markdown
python main.py infosheet mi_afd.json --out mi_afd.md --markdown

# 4) Reporte completo con figuras Graphviz
python main.py report mi_afd.json --out output/mi_afd/

O bien, sin instalar nada, usa los visualizadores HTML de esta página para todo lo que no requiera el aparato algebraico (compilar regex, simular MT). Las hojas informativas precalculadas de los cuatro ejemplos canónicos ya están listas aquí, sin Python.

Referencia bibliográfica

Toda la notación y las definiciones de este sitio siguen el libro base del curso:

De Castro K., R. Introducción a la Teoría de la Computación. Departamento de Matemáticas, Universidad Nacional de Colombia, II Semestre 2024. 219 pp.

Mapeo seccional principal: §2.2 (expresiones regulares), §2.3 (AFD), §2.6 (AFN), §2.8 (AFN-λ), §2.12 (Teorema de Kleene I), §2.13 (Teorema de Kleene II), §2.15 (Myhill-Nerode), §2.16 (minimización), §3.1 (no regularidad), §6.1 (modelo estándar MT), §6.2 (ejemplos canónicos de MT).

El monoide de transición M(A) es una construcción propia del proyecto: no aparece con ese nombre en ninguno de los dos libros. Se apoya en el álgebra de Saracino (monoides, composición, homomorfismos) y en los autómatas de De Castro Korgi. Detalles completos en docs/report.md, sección 12.
